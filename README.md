# Probably a Scam LLC

A cyberpunk storefront theme for a fake scam site selling absurd products like 0GB USB sticks, virus-loaded laptops, and glitchy cables.

## Files

- `index.html` — Main storefront page
- `styles.css` — Cyberpunk dark theme styling
- `script.js` — Report button modal interaction

## Usage

Open `index.html` in a browser to view the site.
